# to-roman-numeral

## Prerequisites

1. [Node v8.9.4](https://nodejs.org/en/download/)
1. [Yarn](https://yarnpkg.com/en/docs/install)

## Installation

```sh
$ git clone git@github.com:ChrisWilding/to-roman-numeral.git
$ cd to-roman-numeral
$ yarn install
$ yarn link
```

## Usage

```sh
$ trn 123
CXXIII
```
